Homely

Homely es una aplicación web desarrollada en Python y Django Framework.
Consiste en una plataforma de búsqueda y contratación de servicios domésticos.

Este proyecto se realizó simulando el nacimiento, progreso, trabajo y lanzamiento de una StartUp, con su posterior presentación ante profesores y algunas empresas que asistieron al evento StartUS, celebrado en la Escuela Técnica Superior de Ingeniería Informática.

Tecnologías: Python 3, Django Framework, Bootstrap, Travis CI, Heroku, Visual Studio Code.
